# Food Order System

## 📌 Project Overview

Food Order System is a simple full-stack web application built using **HTML, CSS, JavaScript, Node.js, and Express.js**. It allows users to place food orders, view all existing orders, and delete orders through an interactive user interface.

The project demonstrates basic CRUD (Create, Read, Delete) operations and frontend-backend communication using REST APIs and the Fetch API.

## ✨ Features

🍔 Place food orders by entering customer name and food item

📋 View all placed orders in a dynamic order list

❌ Delete orders instantly

🔄 Real-time updates without page refresh

✅ Input validation for required fields

📱 Clean and user-friendly interface

⚡ Frontend and backend integration using Fetch API

## 🎨 User Interface

Responsive order form

Customer Name input field

Food Item input field

Order placement button

Dynamic orders list

Delete button for removing orders

Modern card-based layout with CSS styling

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
* Node.js
* Express.js

## 📂 API Endpoints

### Get All Orders

```http
GET /orders
```

### Add New Order

```http
POST /orders
```

Request Body:

```json
{
  "customer": "Ali",
  "food": "Burger"
}
```

### Delete Order

```http
DELETE /orders/:id
```

## 🚀 How to Run

 1. Clone Repository

```bash
git clone https://github.com/urooj624/Food-Order-website.git
```

 2. Install Dependencies

```bash
npm install
```

### 3. Start Server

```bash
node server.js
```

### 4. Open Browser

```text
http://localhost:3000
```
